Math System 은 LaTeX·수식과 수학 글쓰기를 다루는 곳입니다.

## 무엇을 만드나

- **블로그** — LaTeX 문법, 수식 표기 관례, Google Docs·Word 에서 수학 글쓰기 노하우.
- **[LaTeXFlow](/latexflow/web/)** — `.docx` 의 LaTeX 수식 (`$…$` / `$$…$$` / `\(…\)` / `\[…\]` / `[…]`) 을 PNG 이미지로 변환하는 브라우저 도구. Google Docs 애드온도 있습니다.
- **Mathfolis** (개발 중) — 같은 문제가 없는 수학 문제집을 만드는 도구.

도구는 모두 *수학을 다루는 사람* 의 일상 작업 마찰을 줄이는 게 목적입니다.
