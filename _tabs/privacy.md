본 정책은 **mathsystem.dev 사이트** (블로그 및 LaTeXFlow 도구 페이지) 방문 시 수집되는 데이터, 사용되는 쿠키 / 제3자 서비스 / 광고에 대해 안내합니다.

LaTeXFlow 도구 사용 시의 *데이터 처리* (LaTeX 학습 데이터, Add-on 권한 등) 는 [LaTeXFlow Privacy](/latexflow/privacy/) 에서 다룹니다.

## 1. 수집되는 데이터

사이트 방문 시 자동 수집될 수 있는 정보:

- IP 주소 (제3자 서비스 — 아래 §2 — 가 자동 수집)
- 브라우저 정보 (User-Agent, 화면 크기, 언어)
- 방문 페이지 / 시간 / referrer

이 정보는 광고 personalize 및 사이트 분석을 위해 제3자 vendor 가 처리합니다. 본 사이트 운영자가 별도로 저장하지 않습니다.

## 2. 제3자 서비스

| 서비스 | 목적 | 정책 |
|---|---|---|
| Google AdSense | 광고 표시 + personalize | [Google ads 정책](https://policies.google.com/technologies/ads) |
| GitHub Pages | 사이트 호스팅 | [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) |

각 서비스는 자체 Privacy Policy 가 적용됩니다.

## 3. Google AdSense

mathsystem.dev 의 블로그 페이지 및 LaTeXFlow 도구 페이지에는 Google AdSense 광고가 표시될 수 있습니다.

- **쿠키 사용** — Google AdSense 와 그 파트너는 사용자 브라우저에 쿠키 (DoubleClick DART cookie 포함) 를 박아 광고를 personalize 합니다.
- **데이터 전송** — IP 주소, 페이지 방문 정보 등이 Google 서버로 전송될 수 있습니다.
- **상세** — Google 이 파트너 사이트의 데이터를 어떻게 사용하는지는 [Google 파트너 사이트 정책](https://policies.google.com/technologies/partner-sites) 에서 안내됩니다.

## 4. 쿠키 관리

브라우저 설정에서 쿠키를 끄거나 삭제할 수 있습니다:

- [Chrome 쿠키 관리](https://support.google.com/chrome/answer/95647)
- [Firefox 쿠키 관리](https://support.mozilla.org/en-US/kb/clear-cookies-and-site-data-firefox)
- [Safari 쿠키 관리](https://support.apple.com/guide/safari/manage-cookies-sfri11471)
- [Edge 쿠키 관리](https://support.microsoft.com/en-us/microsoft-edge/delete-cookies-in-microsoft-edge)

Google 광고 personalize 만 끄려면 [Google 광고 설정](https://adssettings.google.com).

## 5. EU 사용자 (GDPR)

EU 거주자는 GDPR 에 따른 권리를 가집니다 — 데이터 접근 / 정정 / 삭제 / 처리 거부. Google AdSense 가 EU 사용자에게 별도 consent 페이지를 표시합니다.

## 6. 한국 사용자 (PIPA)

한국 거주자는 PIPA 에 따른 권리 — 개인정보 열람 / 정정 / 삭제 / 처리 정지 — 를 가집니다. 도구 사용 시 데이터 처리는 [LaTeXFlow Privacy §5b](/latexflow/privacy/#5b-web-app--how-to-opt-out) 의 opt-out 절차를 따릅니다.

## 7. 어린이 (13세 미만)

13세 미만 어린이를 대상으로 하지 않습니다. 알게 된 경우 데이터를 삭제합니다.
